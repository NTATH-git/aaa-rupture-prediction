破裂腹部大動脈瘤における 術前壁応力予測と破裂部位の後ろ向き検証

単一症例・後ろ向き解析（匿名化） |  腎下部AAA
術前CT（破裂の約9.5か月前） 破裂 術後（人工血管置換後）

0 概要

同一患者の腎下部腹部大動脈瘤について、破裂の約9.5か月前の待機CTから血流(CFD)・壁応力(FEA)を計算し、 実際の破裂部位（後壁・左・分岐部から約5cm頭側）および術後経過と照合した後ろ向き検証。

要点
  • 高応力域は破裂部位と同じ後面側・同程度の頭尾レベルに存在した。ただし左右の限局性は一致せず、 応力ピークは対側（右後壁）であった（詳細は3節）。壁厚一様・ILT未考慮という本モデルの限界による可能性がある。
  • 破裂前 → 破裂 → 術後の3時点で瘤の増大・破裂・修復を三次元的に可視化。
  • 血流CFDでは瘤中央が低WSS＋高OSI（血栓形成環境）。壁応力（破裂力）と合わせ二面から評価。

臨床経過：2025年5月CTA後、患者本人が経過観察を希望したため待機手術は行われなかった。血管リスク因子は高血圧・高脂血症で、降圧薬およびスタチンを内服していた。

1 自然史 — 大動脈3時点 3D再構成

動脈相CTから大動脈をセグメントし、同一配色・同一視点で比較。瘤の増大 → 破裂 → 人工血管置換が読み取れる。

3 破裂部位との照合

実破裂部位（後壁・左・分岐部+約5cm）を3Dモデル上に同定し、壁応力分布と定量照合した。 分岐部は実測 z≈1124 mm、破裂高位は z≈1174 mm。解剖軸は画像の affine と、独立した左右命名（腎動脈・腸骨動脈）、 臨床VRの L/R マーカーの3系統で検証済み。

照合の結果
  • 一致した点：後面側（後壁側が上位5%応力セルの70.9%／面積比52.6%＝1.35倍の濃縮）と、 頭尾方向のレベル（破裂高位 p95 370 kPa ＞ 嚢全体 351 kPa）。
  • 一致しなかった点：左右の限局性。実際に破裂した左後壁は破裂高位で最も応力が低い象限で、 ピークは対側の右後壁にあった（下表）。
  • 解釈の限界：本モデルは壁厚一様・ILT（壁在血栓）未考慮・線形弾性。本症例は壁在血栓を伴い、 偏在ILTは応力分布を大きく変えることが知られる。この不一致自体がモデルの限界に由来する可能性があり、ILTを含めた再解析が必要。
  • 破裂＝局所応力 ≧ 局所強度。壁応力は「力」のみを表し、壁の「強度」は非侵襲に測れない。応力単独では破裂部位を特定できない。

4 血行力学 CFD

拍動流CFDで TAWSS/OSI/RRT を評価。瘤中央は低WSS＋高OSI＝血栓形成・壁変性の環境。

OSIについての注記
  • 入口波形が三相性（逆流を含む）のため拡張期に壁全体でWSSが反転しOSIが底上げ。生理的には妥当だが局所識別力は乏しい。
  • 破裂部位の局在評価は TAWSS/RRT・壁応力を主指標にするのが適切。

参考：全大動脈CFD（可視化用）

5 方法（詳細）

SimVascular 非依存の自作パイプライン。オープンソース（TotalSegmentator / OpenFOAM / scikit-fem / meshpy）のみで、 セグメンテーション → 幾何・メッシュ → 血流CFD → 壁応力FEA → 破裂照合を再現可能な形で構成した。

5.1 画像・セグメンテーション

  • 造影CT（動脈相）を dcm2niix で DICOM→NIfTI 変換。
  • TotalSegmentator で大動脈・左右腸骨動脈・左右腎動脈ほかを自動抽出し、AAA嚢〜腎ネック〜分岐〜腸骨を統合した解析マスクを作成。

5.2 幾何・メッシュ生成

  • マスクを平滑化し marching cubes で内腔（lumen）表面を再構成、面数をデシメーション（約2mm解像度）。
  • CFD：内腔をそのまま流体領域として体積メッシュ化。
  • FEA（中空壁シェル）：外側（嚢外表面）と内側（内腔表面）の2枚の閉曲面、および内腔内に置いた「穴（hole）」シード点を meshpy / TetGen に渡し、壁の隙間のみを四面体で充填（実効厚 約1.9mm、約110万四面体）。tetgen hole 法

5.3 血流CFD（OpenFOAM pimpleFoam）

非定常・非圧縮 pimpleFoam（OpenFOAM v2512）で、823,279セルの腎下部モデルを3心拍解析し、最終心拍から内腔壁面上の TAWSS/OSI/RRT を算出した。

時間積分1心拍0.9秒（約67 bpm）。3心拍を計算し、最終心拍の周期平均（fieldAverage）で WSS 統計を算出
TAWSS = 時間平均 |WSS|、OSI = 0.5·(1 − |⟨WSSベクトル⟩| / ⟨|WSS|⟩)、RRT = 1 / ((1 − 2·OSI)·TAWSS)。

5.4 壁応力FEA（scikit-fem）

  • 中空壁シェルを線形弾性体としてモデル化（scikit-fem・四面体1次要素）。
  • 材料：E = 2 MPa, ν = 0.45（動脈壁の近似・準非圧縮）。
  • 荷重：内腔面に一様内圧 P = 16 kPa（120 mmHg）。境界条件：モデル両端＋分枝出口を固定。
  • 出力：von Mises／最大主応力。拍動動画は内圧を 80↔120 mmHg で線形スケール（色が拍動・変位ワープなし）。

5.5 破裂照合・可視化

  • 術中所見と破裂時CTで同定した実破裂部位（後壁・左・分岐部+約5cm）を破裂前モデル上に指定し、壁応力・血流指標と空間照合。
  • 可視化は turbo 配色、回転動画に APLR（前後左右）方向インジケーターを付した。

6 限界

  • 壁：厚さ一様1.9mm・ILT未考慮・線形弾性・壁厚方向1要素。
  • 変位：長いモデルの端固定による境界アーティファクト（応力場は有効／組織支持ばねで改善可）。
  • 時間差：解析は破裂の約9.5か月前の幾何。破裂時までの growth は未反映。
  • CFD：汎用入口波形・出口 p=0（RCR未）・Newtonian。全大動脈CFDは可視化用。
  • 単一例・仮説生成的。強度分布（ILT/石灰化）未モデル化のため応力単独の局在予測には限界。
